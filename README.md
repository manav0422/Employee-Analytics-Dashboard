# 📊 HRM Dashboard — Power BI

A Human Resource Management dashboard built with **Microsoft Power BI**, focused on employee attendance, leave management, and performance appraisals.

---

## 🖥️ Overview

This dashboard provides a comprehensive view of key HR metrics to help organizations monitor workforce attendance patterns and evaluate employee performance — all in one interactive interface.

---

## ✨ Features

### 🗓️ Attendance & Leave Tracking
- Daily, weekly, and monthly attendance summaries
- Leave type breakdown (sick, casual, earned, etc.)
- Absenteeism trends over time
- Department-wise attendance comparison

### 🏆 Performance & Appraisals
- Employee performance ratings and distribution
- Appraisal cycle tracking
- Top performers and improvement areas
- Department-level performance benchmarks

---

## 🛠️ Built With

- [Microsoft Power BI Desktop](https://powerbi.microsoft.com/) — Dashboard design & data modelling
- DAX — Custom measures and calculated columns
- Power Query (M Language) — Data transformation

---

## 🚀 Getting Started

### Prerequisites
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free) installed on Windows

### Steps
1. Clone or download this repository
   
   git clone https://github.com/manav0422/Employee-Analytics-Dashboard
   
2. Open `DASHBOARD_hrm.pbix` in Power BI Desktop
3. If prompted, update the data source credentials or file path
4. Refresh the data and explore the report
